# SkyPulse - Real-Time Weather Web Application

## Description
SkyPulse is a responsive weather web application developed using HTML, CSS, and JavaScript that provides real-time weather updates through API integration. Users can search weather conditions by city and view temperature, humidity, wind speed, and weather status through an interactive interface.

## Features
- Real-time weather updates
- Search weather by city
- 5-day weather forecast
- Air quality information
- Current location weather
- Responsive design
- Dynamic weather icons

## Technologies Used
- HTML5
- CSS3
- JavaScript
- OpenWeather API
